
## Задача: Анализ психологического дневника

### Исходные данные

БД создается запуском script.py   
В SQLite базе данных есть таблица `entries` со следующей структурой:

| Поле          | Тип     | Описание                                           |
|---------------|---------|----------------------------------------------------|
| `date`        | TEXT    | Дата записи (формат YYYY-MM-DD), первичный ключ   |
| `mood`        | INTEGER | Оценка настроения от 1 (очень плохо) до 10 (отлично) |
| `sleep_hours` | REAL    | Количество часов сна прошлой ночью                 |
| `activity`    | TEXT    | Основная деятельность за день (work, sport, rest, party, learning, meditation, gaming, walk) |
| `notes`       | TEXT    | Произвольные заметки                               |

### Требуется написать программу на Python, которая:
1. **Читает все данные из таблицы** 
2. **Находит «золотые дни»** – дни, когда настроение > 8 **и** продолжительность сна > 7 часов. Вывести список таких дней с указанием даты, настроения, часов сна и активности.
4. **Определяет, после какой активности на следующий день настроение падает сильнее всего.** Для этого нужно для каждого дня (кроме последнего) вычислить изменение настроения: `mood_завтра - mood_сегодня`, сгруппировать изменения по активности **предыдущего дня**, усреднить и найти активность с минимальным (самым отрицательным) средним изменением.
